# SQL-PowerBI_Project
End-to-end data analytics project using SQL for querying and transformation. Power BI dashboards provide clear and actionable visual insights. Focused on data-driven decision making.

## 1. Healthcare Analysis

This project focuses on analyzing healthcare operational, patient, doctor, appointment, treatment, and revenue data using SQL for data preparation and analysis and Power BI for interactive dashboards. The goal is to derive actionable insights that support hospital management in decision-making, resource planning, and performance monitoring.
